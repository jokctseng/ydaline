Intro

青年發展協進會 Line 社群旨在讓對青年事務與資訊有興趣的大家可以在此分享訊息與交流。社群由青年發展協進會創立，由當屆理事長、秘書長以及公共溝通組長擔任社群管理員，除了青發協會員與社群網站追蹤者外，我們向對青年發展有興趣的朋友開放。如果想認識我們，可以查看 About區塊 或者是我們的 官方網站.

關於青發協社群特典行事曆或其他特典，請查看 Specials 區塊

此外我們也想向您說明社群規範，其中多數來自於 Line 為社群功能提出的規範，若有違反平臺或社群規定，我們會先予以警告，情節嚴重或屢勸未改者將踢出社群，還請見諒。這些規範您隨時可以在社群的記事本中查看

  • 歡迎交流青年事務與資訊
  • 歡迎閒聊
  • 禁止發放政黨文宣（政府官方宣導與活動不在此限）
  • 禁止人身攻擊與仇恨言論
  • 禁止在聊天內容中揭露個人Line ID
  • 禁止在 Line 社群中溝通有「單獨會面意圖」的所有對話
  • 禁止傳直銷相關討論
  • 禁止暴力、血腥、色情、恐怖等有害兒少身心健康相關討論與內容
  • 禁止交易討論
  • 禁止非經允許發放具商業、營利性質廣告或問候圖
  • 禁止其他任何違法行為

關於青年發展協進會

青年發展協進會，源於2017年底，一群來自各地的青年諮詢委員聚集交流時，想改變政府對青年認知不一的狀況，期望凝聚青年意識與認知共識，並於2018年正式立案成立，最終目標為促成能保障青年基本權利的青年發展法，在推動立法前，也將先提出對青年權利與發展的願景、論述與活動

若想查看平時如何聯繫我們，請看 Contact區塊
